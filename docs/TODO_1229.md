### **쉽게 이해하는 신호등 로직** ⭐

**📖 비유로 설명:** 경제 상황을 "건강 검진"이라고 생각해보세요!

**1단계: 각 지표별 "건강 점수" 매기기** (점수 계산)

- **장단기 금리차 (spread)**: 경제의 "혈압" 같은 것

  - -0.5 이하면: 위험! (+3점)
  - -0.3~-0.5: 주의 (+2점)
  - -0.1~-0.3: 보통 (+1점)
  - 0 이상: 건강 (+0점)

- **미국 실업률 (unrate)**: 경제의 "체온" 같은 것

  - 4.5% 이상: 열이 높음! (+2점)
  - 4.0~4.5%: 약간 높음 (+1점)
  - 4.0% 미만: 정상 (+0점)

- **환율 (exchange)**: 경제의 "맥박" 같은 것
  - 1500원 이상: 너무 빠름! (+2점)
  - 1400~1500원: 약간 빠름 (+1점)
  - 1400원 미만: 정상 (+0점)

**2단계: 총점 계산해서 신호등 색상 결정**

- 총점 5점 이상: 🔴 빨강 (위험! 경제 위기 가능성 높음)
- 총점 3~4점: 🟡 노랑 (주의! 경제 상황 주시 필요)
- 총점 0~2점: 🟢 초록 (안전! 경제 상황 양호)

**3단계: 테스트하기** (실제 데이터로 검증)

```javascript
// 예시: 실제 경제 상황 테스트
const 실제데이터 = {
  spread: -0.5, // 장단기 금리차: -0.5% (위험!)
  unrate: 4.2, // 실업률: 4.2% (주의)
  exchange: 1450, // 환율: 1450원 (주의)
};
// 계산 결과: 3+1+1 = 5점 → 🔴 빨강 신호!
```
