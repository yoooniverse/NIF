# 프로젝트 수행 경과 (단계별)

## 1. 경제 뉴스 해설 수행 경과

### Step 1: 뉴스 수집 파이프라인 구축 (Week 1, Day 3-5)
- n8n 워크플로우로 RSS 피드에서 경제 뉴스 자동 수집
- 어제 날짜 뉴스만 필터링하여 DB에 저장

### Step 2: AI 해설 생성 시스템 개발 (Week 1, Day 3-5)
- Google Gemini 2.5 Flash API 연동
- 관심분야별(주식/부동산/ETF 등) 뉴스 분류 및 해설 생성
- 레벨별(초보/일반/전문가) 해설 자동 생성 (1개 뉴스 → 3개 레코드)

### Step 3: 개인화 시나리오 생성 (Week 1, Day 3-5)
- 사용자 상황별(직장인/대출보유/예적금 등 6가지) 최악 시나리오 생성
- 각 상황별 액션 팁 자동 생성
- news_analysis_levels 테이블에 저장

### Step 4: 프론트엔드 연동 (Week 2, Day 11-13)
- 뉴스 목록 페이지 구현 (관심분야별 필터링)
- 뉴스 상세 페이지 구현 (AI 해설 + 최악 시나리오 + 액션 팁)
- 사용자 선택한 레벨/상황에 맞는 콘텐츠만 표시

### Step 5: 자동화 완성 및 검증 (Week 2, Day 14)
- n8n 워크플로우 매일 자동 실행 설정 완료
- 실제 뉴스 데이터로 전체 플로우 검증 완료

## 2. 경제 상황 브리핑 (경제 순환기 지도) 수행 경과

### Step 1: 데이터 소스 연동 (Week 3, Day 17-18)
- FRED API 연동하여 경제 지표 수집
  - 장단기 금리차 (T10Y2Y)
  - 미국 실업률 (UNRATE)
  - 원/달러 환율 (DEXKOUS)

### Step 2: 신호등 색상 로직 구현 (Week 3, Day 17-18)
- 규칙 기반 알고리즘으로 위험도 점수 계산
- Red/Yellow/Green 신호등 색상 자동 결정
- determineStatusColor 함수 구현

### Step 3: AI 요약 생성 시스템 (Week 3, Day 19)
- Claude 3.5 Sonnet API 연동
- 신호등 색상 기반 현재 상황 요약 자동 생성
- 역사적 패턴/팩트 자동 작성

### Step 4: 데이터베이스 및 API 구축 (Week 3, Day 19)
- cycle_explanations 테이블 생성
- GET /api/cycle/current API 구현
- 최신 데이터 조회 로직 구현

### Step 5: 프론트엔드 구현 (Week 3, Day 20-21)
- 경제 순환기 페이지 (/cycle) 구현
- 좌측: 순환기별 특징 표 (회복기/확장기/둔화기/침체기)
- 우측: 근거 지표 시각화 (신호등 + 레이더 차트 + 계기판)
- 대시보드 모달 연동

### Step 6: 자동화 워크플로우 완성 (Week 3, Day 20-21)
- n8n 워크플로우로 매일 자동 업데이트 설정
- FRED API → 신호등 계산 → AI 요약 → DB 저장 자동화 완료
- 2026.01.06 최종 완성 ✅
