# News In Flight - PPT 요약

## 프로젝트 주제의 특화 포인트

**AI 기반 개인화 경제 뉴스 해석 서비스**

- **개인 상황 맞춤 해석**: 사용자 선택(관심분야, 상황, 레벨)에 따라 뉴스를 개인화된 언어로 해석
- **경제 순환기 시각화**: 현재 경제 위치를 3D 지구와 신호등 색상으로 직관적 표현
- **최악 시나리오 + 액션 가이드**: 뉴스가 내 자산에 미치는 영향과 구체적 행동 지침 자동 생성
- **In Flight 컨셉**: 비행기 여행 메타포를 활용한 프리미엄 UX (3D 지구 배경, Boarding Pass 모달)

**기존 서비스와의 차별화**
- 어피티/뉴닉: 일반 정보 전달 → **개인 상황별 시나리오 제공**
- 텍스트 중심 → **시각화 중심 (경제 시계, 사이클 맵)**
- 수동 큐레이션 → **AI 자동 분석 파이프라인**

---

## 프로젝트 내용

**구현 컨셉**
경제 뉴스를 "내 언어"로 번역하고, 경제 사이클 상의 위치를 시각화하여 자산 보호 및 증식 판단을 돕는 플랫폼

**주요 구현 기능**
1. **온보딩 시스템**: 관심 자산(주식/부동산/ETF 등), 나의 상황(대출보유/직장인 등), AI 레벨(초보/일반/전문가) 선택
2. **AI 뉴스 파이프라인**: n8n + Google Gemini로 매일 자동 뉴스 수집 및 해설 생성 (레벨별, 상황별)
3. **3D 대시보드**: Three.js 기반 지구 배경, "오늘의 뉴스", "이달의 뉴스", "경제 순환기 지도" 제공
4. **뉴스 상세**: AI 해설 + 최악 시나리오 + 액션 팁 (30일 무료 체험 후 Paywall)
5. **경제 순환기 지도**: FRED API 연동, 신호등 색상(Red/Yellow/Green) 자동 계산, AI 요약 생성

**훈련 내용과의 연관성**
- **풀스택 개발**: Next.js 15 (프론트) + Supabase (백엔드) + n8n (자동화)
- **AI 활용**: Gemini API를 통한 자연어 처리 및 콘텐츠 생성
- **모던 웹 기술**: React 19, TypeScript, Tailwind CSS v4, Three.js
- **인증/결제**: Clerk 인증, 구독 시스템 설계

## 활용 장비 및 재료

**개발 환경**
- **프론트엔드**: Next.js 15.5.9, React 19, TypeScript, Tailwind CSS v4
- **백엔드**: Supabase (PostgreSQL), Next.js API Routes
- **인증**: Clerk (소셜 로그인, 이메일)
- **AI**: Google Gemini 2.5 Flash (뉴스 분석), Claude 3.5 Sonnet (경제 순환기 요약)
- **자동화**: n8n (뉴스 수집, AI 분석 트리거, DB 저장)
- **데이터 소스**: FRED API (경제 지표)
- **3D 라이브러리**: Three.js, @react-three/fiber
- **UI 컴포넌트**: shadcn/ui, Radix UI
- **호스팅**: Vercel (프론트/API), Supabase Cloud (DB)

**개발 도구**
- **패키지 매니저**: pnpm
- **버전 관리**: Git
- **에디터**: Cursor (AI 코딩 어시스턴트)

## 프로젝트 구조

**프로젝트 목적**
2030 직장인 및 개인 투자자를 대상으로, 경제 뉴스를 개인 상황에 맞게 해석하여 자산 보호 및 증식 판단을 지원하는 AI 기반 인사이트 플랫폼 구축

**진행 일정**
- **Week 1-2 (Day 1-14)**: 인증, 온보딩, AI 파이프라인, 대시보드, 뉴스 상세 ✅
- **Week 3 (Day 15-21)**: 경제 순환기 지도, n8n 워크플로우 완성 ✅
- **Week 4-5 (Day 22-33)**: QA, 최적화, 배포 준비 🔄

**구체적 실행 내역**
- **데이터베이스**: users, user_profiles, news, news_analysis_levels, cycle_explanations 등 10+ 테이블 설계
- **API**: 뉴스 조회, 경제 순환기 조회, 구독 상태 확인 등 RESTful API 구현
- **자동화**: n8n 워크플로우로 매일 뉴스 수집 → AI 분석 → DB 저장 자동화
- **UI/UX**: 3D 지구 배경, 반응형 디자인, 다크모드 지원, 접근성 고려

## 활용방안 및 기대 효과

**프로젝트 산출물의 기대 효용**

1. **정보 비대칭 해소**: 어려운 경제 뉴스를 개인 상황에 맞게 쉽게 해석
2. **즉각적 대응책 마련**: 최악 시나리오와 액션 가이드를 통한 실용적 인사이트 제공
3. **경제 흐름 파악**: 경제 순환기 시각화로 큰 그림(Big Picture) 이해
4. **시간 절약**: 여러 채널 탐색 없이 한 곳에서 개인화된 정보 제공

**비즈니스 실무 활용성**

- **수익 모델**: 30일 무료 체험 → 월 5,900원(얼리버드) / 9,900원 구독
- **확장 가능성**: 초기 경제 뉴스 해석 → 장기적으로 로보어드바이저 플랫폼으로 확장
- **시장 기회**: 국내 25~39세 경제활동인구 약 1,000만 명 중 유료 전환 가능층 약 8만 명 타겟
- **기술 재사용**: AI 파이프라인, 개인화 시스템, 구독 모델 등 다른 도메인에도 적용 가능

**기대 효과**
- **사용자**: 경제 정보 접근성 향상, 자산 관리 의사결정 지원
- **비즈니스**: 유료 구독 전환율 15%+ 목표, 연간 ARR 95억 원 목표
- **기술**: AI 기반 콘텐츠 생성, 자동화 워크플로우 구축 경험
