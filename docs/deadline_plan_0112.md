## ⚠️ 리스크 관리

### High Risk

- **경제 순환기 지도:** 현재 50% 진행 중, 12/30까지 완료 필수
- **n8n 워크플로우:** API 연동 복잡성으로 인한 지연 가능성

### Medium Risk

- **구독 시스템:** MVP로 축소하되 UI만 구현
- **QA 시간 부족:** 병렬 QA 진행으로 해결

### Mitigation Strategy

1. **현재 진행 중인 작업 우선 완료**
2. **구독 시스템은 추후 업데이트로 미룸**
3. **QA는 핵심 기능만 집중**
4. **배포는 1/11부터 준비 시작**
5. **프로필 설정 페이지 구현 난이도 낮음** ✅ (관심분야 재설정 기능 웹에서 구동 확인)
